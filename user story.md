As a ลูกค้าสายเกมเมอร์

I want กรองหา RAM ที่มีไฟ RGB และดูว่ารองรับระบบซิงค์ไฟของเมนบอร์ดรุ่นที่ฉันมีหรือไม่ (เช่น Aura Sync, RGB Fusion)

So that ฉันสามารถแต่งเคสคอมพิวเตอร์ให้มีความสวยงามและควบคุมแสงไฟให้ไปในทิศทางเดียวกันได้

Given ลูกค้าอยู่ในหน้าค้นหาสินค้า RAM

When ลูกค้าติ๊กเลือกฟิลเตอร์ "RGB Sync" และเลือกแบรนด์ "ASUS"

Then ระบบจะแสดงเฉพาะสินค้า RAM ที่มีไฟ RGB และมีสัญลักษณ์รองรับ ASUS Aura Sync เท่านั้น


As a พนักงานจัดการสต็อกสินค้าร้านคอมพิวเตอร์

I want สแกนบาร์โค้ด (Serial Number) บนกล่อง RAM เพื่อตัดสต็อกทันทีที่มีการขายออกหน้าร้าน

So that สต็อกสินค้าในระบบออนไลน์และหน้าร้านจะตรงกันแบบเรียลไทม์ ป้องกันปัญหาลูกค้าออนไลน์กดสั่งซื้อสินค้าที่หมดไปแล้ว

Given พนักงานเข้าสู่ระบบจัดการหลังบ้าน (POS) และมีสินค้า RAM ที่ลูกค้ากำลังจะซื้ออยู่บนมือ

When พนักงานใช้เครื่องสแกนบาร์โค้ด สแกนที่กล่อง RAM และกด "ยืนยันการขาย"

Then ระบบทำการตัดสต็อกสินค้ารายการนั้นออก 1 ชิ้นทันที พร้อมอัปเดตจำนวนสินค้าที่แสดงบนหน้าเว็บไซต์ให้ลดลงตามจริง


As a ลูกค้าทั่วไปที่ไม่มีความรู้เรื่องสเปคคอมพิวเตอร์

I want ถ่ายรูปสเปคเครื่องเดิมหรือตัว RAM เก่าส่งให้พนักงานดูผ่านระบบแชทของร้าน

So that ฉันจะได้รับคำแนะนำและซื้อ RAM รุ่นที่ถูกต้องไปอัปเกรดเครื่องโดยไม่ต้องกลัวว่าจะซื้อผิดสเปค

Given ลูกค้าอยู่ในหน้าแรกของเว็บไซต์ร้าน

When ลูกค้ากดปุ่ม "ปรึกษาช่าง/ผู้เชี่ยวชาญ" และทำการอัปโหลดรูปภาพ RAM ตัวเก่าลงไป

Then ระบบเปิดหน้าต่างแชทและส่งรูปภาพนั้นไปยังหน้าจอของพนักงานที่ออนไลน์อยู่ พร้อมแจ้งเตือนพนักงานให้ตอบกลับภายใน 1 นาที


As a ครีเอเตอร์สายตัดต่อวิดีโอความละเอียดสูง

I want เลือกดูเฉพาะ RAM ที่ขายเป็นเซ็ตคู่ หรือเซ็ต 4 แถว (Kit of 2 / Kit of 4) ที่มีความจุรวม 64GB ขึ้นไป

So that ฉันมั่นใจได้ว่า RAM ทุกตัวจะทำงานแบบ Dual/Quad Channel ได้อย่างเสถียรเมื่อต้องเรนเดอร์งานกราฟิกหนักๆ

Given ลูกค้าเข้ามาที่หมวดหมู่ "RAM สำหรับคนทำงาน (Workstation)"

When ลูกค้าเลือกความจุ "64GB ขึ้นไป" และรูปแบบแพ็คเกจ "Kit (ขายเป็นเซ็ต)"

Then ระบบจะซ่อน RAM ที่ขายแยกแบบแถวเดี่ยว (Single Module) ทั้งหมด และแสดงเฉพาะ RAM ที่แพ็คคู่หรือแพ็ค 4 ที่สเปคตรงกันจากโรงงาน

As a ผู้ใช้งานระดับโปรที่ชื่นชอบการดันสเปคเครื่อง (Overclocker)

I want ดูรายละเอียดเชิงลึกของ RAM เช่น ค่า Timing (CL) อย่างละเอียด และชนิดของชิปหน่วยความจำ (เช่น Samsung B-Die, SK Hynix) บนหน้าเว็บ

So that ฉันสามารถนำข้อมูลไปประเมินประสิทธิภาพในการทำโอเวอร์คล็อกได้ก่อนที่จะตัดสินใจจ่ายเงินซื้อรุ่นที่มีราคาแพง

Given ลูกค้าอยู่ที่หน้ารายละเอียดสินค้า RAM ระดับ Hi-End

When ลูกค้าคลิกขยายแท็บ "ข้อมูลทางเทคนิคเชิงลึก (Advanced Tech Specs)"

Then ระบบแสดงตารางสเปคที่ระบุค่า Timing พื้นฐาน (เช่น CL16-18-18-38), ค่า Voltage และระบุยี่ห้อชิปของ RAM รุ่นนั้นอย่างชัดเจน
